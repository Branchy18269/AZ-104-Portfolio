Lab 05: Intersite Connectivity
Project: Self Guided AZ104 - Using Done Right Tech Company Resources
Objective
Establish secure management and hybrid connectivity paths. By implementing Azure Bastion, we eliminate the need for public RDP/SSH ports. The Virtual Network Gateway prepares the environment for secure encrypted tunnels.

Technical Tasks
Specialized Subnetting: Creating the mandatory GatewaySubnet and AzureBastionSubnet.

Secure Remote Management: Deploying Azure Bastion for SSL-encrypted, browser-based VM access.

Hybrid Connectivity: Provisioning a VpnGw1 SKU gateway for Site-to-Site (S2S) capabilities.
