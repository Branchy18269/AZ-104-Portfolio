Lab 06: Network Traffic Management
Project: Self Guided AZ104 - Using Done Right Tech Company Resources
Objective
Implement high-availability networking for DRTC applications. A Standard Azure Load Balancer is used to provide a single entry point that intelligently distributes traffic and monitors backend health.

Technical Tasks
Public Load Balancer: Provisioning a Standard SKU LB with a static frontend IP.

Health Monitoring: Configuring a TCP health probe on Port 80.

Traffic Orchestration: Mapping incoming traffic to the backend pool via Load Balancing Rules.
